# webhoster
local web hosting program using c++ and Qt
----------------------------------------
ðŸ“Œ How to Use

1. Copy WebHoster.exe into the folder that contains your website files.
2. Run the application.
3. Click Start Server and allow the required network permissions.
4. To find your IP, open Command Prompt and type:
   <h1>[CMD] >ipconfig</h1>
5. If no network is available, WebHoster uses the loopback address: <h1>127.0.0.1</h>
6. The server will automatically load your index.html file (make sure it exists in the same folder).
